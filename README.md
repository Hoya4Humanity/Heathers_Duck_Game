# Heather’s Duck Game 🪠🦆

Play it live:
https://hoya4humanity.github.io/Heathers_Duck_Game/

## Controls
- Click/tap: 🪠 BONK
- Double-click / double-tap: 🧻 BOINK (TP roll, cooldown)
- Space: pause/resume
- R: reset run
- M: mute/unmute

## Troubleshooting
If you see a README-like page instead of the game:
- You do **not** have a valid root `index.html` starting with `<!doctype html>`.
- Ensure `index.html` exists at the repo root and is real HTML (not markdown).
- `.nojekyll` helps ensure static files are served directly.
